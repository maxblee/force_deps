force-dependencies
===============================

version number: "0.1.0"
author: Max Lee

Overview
--------

Manage dependencies at the function- or class-level.

Installation / Usage
--------------------

To install use pip:

    $ pip install force_deps


Or clone the repo:

    $ git clone https://github.com/maxblee/force_deps.git
    $ python setup.py install
